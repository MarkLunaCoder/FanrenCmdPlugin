# fanren 资料整理

本目录内容整理自 [iwooji77/fanren](https://github.com/iwooji77/fanren)，并为本项目的指令面板做了本地化整理。感谢原项目整理和维护凡人修仙文字游戏相关资料。本目录不是上游仓库的 Git submodule，也不保证与上游当前 `main` 分支逐字一致。

本目录及由本目录打包进 APK assets 的指令资料不纳入项目根目录 MIT 授权范围；如相关权利人认为不适合继续公开展示或随 APK 分发，请通过 GitHub Issue 联系维护者删除或调整。

如需跟进上游更新，请先对比上游变更，再同步整理 `xiuxian_game_commands.md`，并运行项目测试与构建验证。具体对比命令见项目根目录 README 的“资料来源与更新”章节。

原始资料介绍的是一个基于 Telegram 群聊的长线文字修仙游戏。玩家从检测灵根开始，靠闭关、宗门、洞府、交易、斗法、副本和事件逐步提升境界。游戏具有较强的异步特征：很多收益来自每日命令、后台结算、限时队伍和随机奇遇。

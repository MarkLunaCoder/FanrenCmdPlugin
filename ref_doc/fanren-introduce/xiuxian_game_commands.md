# 修仙游戏命令整理

本文整理项目中修仙游戏相关命令。命令通常以 `.` 开头；`<...>` 表示必填参数，`[...]` 表示可选参数。

## 入门与角色

| 命令 | 用途 |
| --- | --- |
| `.帮助` | 查看基础帮助。 |
| `.检测灵根` | 开始修仙/检测灵根。 |
| `.强行修仙 [@用户或ID]` | 强制为自己或指定用户进入修仙流程。 |
| `.我的灵根` / `.他的灵根 @用户` | 查看自己或他人的修士面板。 |
| `.状态` / `.我的状态` | 查看当前虚弱、避世、阵法等状态。 |
| `.我的道号` / `.他的道号 <用户>` | 查看道号。 |
| `.改道号 <新道号>` | 修改道号。 |
| `.我的奇缘` / `.他的奇缘 <用户>` | 查看奇缘信息。 |
| `.生平 [@用户]` | 生成修士生平。 |
| `.切换 [角色]` | 切换当前操控角色。 |
| `.储物袋` | 查看背包、材料、道具。 |
| `.学习 <功法或配方>` | 学习功法、配方或可学习物。 |
| `.服用 <物品> [数量]` / `.使用 <物品> [参数]` | 使用丹药、符箓或特殊道具。 |
| `.赠送 <物品> <数量>` | 向回复对象或指定目标赠送物品。 |
